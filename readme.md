# Flower Shop Website

A simple and responsive flower shop website built using **HTML, CSS, and JavaScript**.

## Live Features
http://127.0.0.1:5500/index.html

- Responsive navigation bar
- Hero/banner section
- Plants collection section
- Flower & Plants Lover section
- Latest Deals section
- Newsletter join section
- Footer with menu and social icons
- "Add to Cart" alert button using JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript
